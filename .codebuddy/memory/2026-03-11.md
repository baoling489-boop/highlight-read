# 2026-03-11 工作日志

## 修复 epubjs themes.override 调用方式
- **问题**：字号、行间距、页边距调节无效
- **根因**：`themes.override('font-size', '18px !important')` 中 `!important` 被 `setProperty` 当成值的一部分，导致 CSS 值无效。`setProperty` 需要通过第三个参数 `"important"` 来设置优先级
- **修复**：将所有 `themes.override` 调用中的 `!important` 从值中移除，改为通过第三个参数 `priority=true` 传递
- **影响范围**：font-size、line-height、padding、color、background 五处 override 调用
- **文件**：`src/components/EpubReader.tsx` 第 527-555 行
