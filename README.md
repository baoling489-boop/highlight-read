# 📖 EPUB 智能高亮阅读器

> 一款运行在浏览器中的 EPUB 电子书阅读器，支持自定义多彩高亮标注、ADHD 专注阅读模式和阅读时间统计。

🔗 **在线体验**：[https://baoling489-boop.github.io/highlight-read/](https://baoling489-boop.github.io/highlight-read/)

---

## ✨ 功能特性

### 📚 EPUB 阅读
- 支持拖拽或点击上传 `.epub` 格式电子书
- 自动解析书籍目录，侧边栏导航，快速跳转章节
- 翻页支持键盘（← → 空格）、按钮、左右热区点击三种方式
- 实时显示当前章节标题和阅读进度

### 🎨 多彩自定义高亮
- 阅读中选中任意文字，弹出颜色选择器，一键标记
- 提供 **10 种预设颜色**：蓝、绿、黄、橙、紫、红、青、粉、靛蓝、棕
- 侧边栏「我的高亮」面板统一管理：可更换颜色、删除
- 高亮词按书籍独立存储，切换不同书互不干扰
- 支持高亮词的 **JSON 导出/导入**，方便分享和备份

### 🧠 ADHD 专注模式
专为注意力不集中的读者设计，4 个子功能可独立开关：

| 功能 | 说明 |
|------|------|
| **句首/句尾强化** | 每句话的前几个字渐变加粗，末尾几个字渐变蓝色，引导视线流动 |
| **聚焦行高亮** | 鼠标所在行高亮、上下区域调暗，形成"聚光灯"效果 |
| **段首强化** | 每段开头 20 字加粗，帮助快速定位新段落 |
| **字间距增强** | 增加字间距和词间距，提升文字辨识度 |

### ⏱ 阅读时间统计
- 自动统计本次阅读时长和累计阅读时长
- 数据自动保存，关闭后下次打开继续累计

### 🎛 可读性自定义
- **字号调节**：12px ~ 28px
- **行间距调节**：1.4 ~ 3.0
- **页边距调节**：10px ~ 80px
- **明暗主题切换**：柔和纸质色 / 深色护眼模式

## 🚀 本地运行

### 环境要求
- Node.js ≥ 16
- npm 或 pnpm

### 步骤

```bash
# 克隆仓库
git clone https://github.com/baoling489-boop/highlight-read.git
cd highlight-read

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

浏览器访问 `http://localhost:5173` 即可使用。

### 构建部署

```bash
npm run build
```

产物在 `dist/` 目录，可部署到任意静态托管平台（GitHub Pages、Vercel、Netlify 等）。

## 📝 使用指南

1. **打开书籍** — 拖拽 `.epub` 文件到页面，或点击上传区域选择文件
2. **阅读翻页** — 点击左右区域 / 键盘 ← → / 工具栏翻页按钮
3. **添加高亮** — 选中想要标记的文字（1~20字），在弹出的颜色面板中选择一种颜色
4. **管理高亮** — 点击侧边栏「🎨 我的高亮」标签，查看、修改颜色或删除
5. **导出/导入** — 在高亮面板底部，点击导出保存为 JSON，或导入他人分享的高亮文件
6. **ADHD 模式** — 点击工具栏「ADHD」按钮开启，点击齿轮图标自定义子选项
7. **调节阅读** — 工具栏上直接调整字号、行间距、页边距，或切换明暗主题
