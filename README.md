# 📖 EPUB 智能高亮阅读器

> 一款运行在浏览器中的 EPUB 电子书阅读器，支持自定义多彩高亮标注、ADHD 专注阅读模式和阅读时间统计。

🔗 **在线体验**：[https://baoling489-boop.github.io/highlight-read/](https://baoling489-boop.github.io/highlight-read/)

---

## ✨ 功能特性

### 📚 EPUB 阅读
- 支持拖拽或点击上传 `.epub` 格式电子书
- 自动解析书籍目录，侧边栏导航，快速跳转章节
- 滚动阅读模式，支持键盘（↑↓ 平滑滚屏、← → 翻章、Space/PageDown 翻页）、按钮、左右热区点击
- 实时显示当前章节标题和阅读进度（基于全书字数精确计算）

### 🎨 多彩自定义高亮
- 阅读中选中任意文字，弹出颜色选择器，一键标记
- 提供 **10 种预设颜色**：蓝、绿、黄、橙、紫、红、青、粉、靛蓝、棕
- 侧边栏「我的高亮」面板统一管理：可更换颜色、删除
- 高亮词按书籍独立存储，切换不同书互不干扰
- 支持高亮词的 **JSON 导出/导入**，方便分享和备份

### 🧠 ADHD 专注模式
专为注意力不集中的读者设计，4 个子功能可独立开关：

| 功能 | 说明 |
|------|------|
| **句首/句尾强化** | 每句话的前几个字渐变加粗，末尾几个字渐变蓝色，引导视线流动 |
| **聚焦行高亮** | 鼠标所在行高亮、上下区域调暗，形成"聚光灯"效果 |
| **字间距增强** | 增加字间距和词间距，提升文字辨识度 |
| **行间距增强** | 加大行间距，降低行间视觉干扰 |

### ⏱ 阅读时间统计
- 自动统计本次阅读时长和累计阅读时长
- 数据自动保存，关闭后下次打开继续累计

### 🔖 书签系统
- 一键添加/移除当前页面书签
- 侧边栏统一管理，快速跳转到书签位置
- 按书籍独立存储，切换书互不干扰

### 🎛 可读性自定义
- **字号调节**：12px ~ 28px
- **明暗主题切换**：柔和纸质色 / 深色护眼模式，切换有柔和过渡动画

### 📖 阅读体验优化
为获得最佳阅读体验，内置了多项专业排版和交互优化：

| 类别 | 优化内容 |
|------|---------|
| **排版** | 最大阅读宽度 42em（每行 35-42 字），标题字号递减层次，段间距与字号联动，响应式内边距 |
| **字体** | 优先使用衬线体，回退策略覆盖 macOS/Windows 系统中文字体（思源宋体 → 宋体 SC → 宋体） |
| **视觉** | 柔和文字对比度（非纯黑/纯白），暗色模式下链接/引用/代码块全适配 |
| **滚动** | 细滚动条（6px 圆角半透明），`smooth scroll` 平滑滚动，键盘翻页滚动 85% 视口 |
| **交互** | 左右热区 hover 渐变+箭头提示，工具栏 3 秒无操作自动隐藏（鼠标到顶部自动滑出） |
| **混排** | 中英文混排自动换行，长 URL 不撑破布局 |

## 🚀 本地运行

### 环境要求
- Node.js ≥ 16
- npm 或 pnpm

### 步骤

```bash
# 克隆仓库
git clone https://github.com/baoling489-boop/highlight-read.git
cd highlight-read

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

浏览器访问 `http://localhost:5173` 即可使用。

### 构建部署

```bash
npm run build
```

产物在 `dist/` 目录，可部署到任意静态托管平台（GitHub Pages、Vercel、Netlify 等）。

## 📝 使用指南

1. **打开书籍** — 拖拽 `.epub` 文件到页面，或点击上传区域选择文件
2. **阅读翻页** — `↓/Space/PageDown` 平滑滚动一屏，`↑/PageUp` 回滚一屏，`←/→` 翻章，也可点击左右热区
3. **添加高亮** — 选中想要标记的文字（1~20字），在弹出的颜色面板中选择一种颜色
4. **管理高亮** — 点击侧边栏「🎨 我的高亮」标签，查看、修改颜色或删除
5. **导出/导入** — 在高亮面板底部，点击导出保存为 JSON，或导入他人分享的高亮文件
6. **添加书签** — 点击工具栏书签图标标记当前位置，侧边栏管理所有书签
7. **ADHD 模式** — 点击工具栏「ADHD」按钮开启，点击齿轮图标自定义子选项
8. **调节阅读** — 工具栏上直接调整字号，或切换明暗主题
9. **沉浸阅读** — 阅读时工具栏 3 秒后自动隐藏，鼠标移到顶部即可唤出
